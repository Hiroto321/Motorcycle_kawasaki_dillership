# 🏍️ Kawasaki Dealership

Сайт о мотоциклах Kawasaki с удобным интерфейсом и поиском.

## 📖 О проекте

**Kawasaki Dealership** — это веб-приложение для мотосалона, которое позволяет:

## ✨ Возможности

- 📂 Просмотр каталога мотоциклов
- 👤 Личный кабинет
- 🚚 Калькулятор стоимости доставки
- Просмотр статистики

## 🛠 Технологический стек

- **Backend:** Python, Flask, FastAPI
- **Frontend:** HTML, CSS, JavaScript
- **База данных:** PostgreSQL
